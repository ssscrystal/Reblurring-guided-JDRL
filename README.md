Python >= 3.7

PyTorch >= 1.7

You can download the pre-trained weights of SDD/DPDD/DED training datasets from [this link](https://pan.baidu.com/s/1bzapKMoCEt0Hf4MWvvl-3A?pwd=7ttz).
